# IBM MQ Bootcamp

Enterprise messaging bootcamp for IBM MQ architecture, administration and operations.

## Overview

This bootcamp is designed to help architects, administrators, integration specialists and platform teams understand, operate and modernize enterprise messaging environments using IBM MQ.

IBM MQ is a foundational technology for mission-critical integration. This repository will evolve as a practical knowledge base with concepts, labs, operational patterns and reference architectures.

## Who Is This For?

- Middleware Administrators
- Integration Specialists
- Enterprise Architects
- Platform Engineering Teams
- DevOps Teams
- Application Support Teams
- Technology Leaders responsible for mission-critical systems

## Learning Modules

### Module 01 - IBM MQ Fundamentals

Core concepts of enterprise messaging, queue managers, queues, channels, listeners, messages and basic MQ architecture.

### Module 02 - Administration and Operations

Queue manager administration, object management, monitoring and operational routines.

### Module 03 - Secure Operations

Secure connectivity, access control concepts, certificates and enterprise messaging protection practices.

### Module 04 - Troubleshooting

Common failure scenarios, logs, error codes, channel issues, application connectivity and diagnostic practices.

### Module 05 - High Availability and Disaster Recovery

Architecture patterns for resilience, availability, backup, recovery and enterprise continuity.

### Module 06 - Best Practices and Modernization

Operational excellence, automation, containers, Kubernetes, OpenShift and modernization strategies for IBM MQ environments.
