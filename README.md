# IBM MQ Bootcamp

Enterprise Messaging, Administration, Operations and Modernization with IBM MQ

---

## Overview

The IBM MQ Bootcamp is a comprehensive training program designed to help architects, administrators, integration specialists and platform engineering teams master IBM MQ in enterprise environments.

Unlike traditional vendor-focused training, this bootcamp emphasizes real-world operations, administration, troubleshooting, resilience and modernization practices used in production environments.

Students learn not only how IBM MQ works, but how IBM MQ is operated, monitored, secured and modernized across mission-critical enterprise workloads.

## Learning Journey

### Module 01 - Fundamentals

Enterprise messaging concepts, IBM MQ architecture, messaging patterns, delivery guarantees and core object model.

### Module 02 - Administration & Operations

Queue Manager administration, MQSC, queues, channels, listeners, monitoring, backup and operational best practices.

### Module 03 - Security

Authentication, authorization, CHLAUTH, TLS, certificates and secure enterprise messaging.

### Module 04 - Clustering

IBM MQ clustering architecture, workload balancing, repositories and cluster administration.

### Module 05 - High Availability

Native HA, Multi-Instance Queue Managers, failover architectures and operational resilience.

### Module 06 - Disaster Recovery

Backup strategies, recovery procedures, recovery objectives and business continuity planning.

### Module 07 - Performance & Troubleshooting

Diagnostic techniques, FDC analysis, log interpretation, common MQRC codes and operational troubleshooting.

### Module 08 - Cloud Native MQ

Containers, Kubernetes, OpenShift, Operators and Cloud Pak for Integration.

### Module 09 - Modernization Patterns

Hybrid integration architectures, event-driven modernization, API integration and enterprise modernization strategies.
